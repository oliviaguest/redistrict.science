Welcome

This is the website accompanying the manuscript Gerrymandering and Computational Redistricting by Olivia Guest, Frank J. Kanayet, and Bradley C. Love.

For the materials used, please see the Open Framework Repository, here: osf.io/8xpq4. This repository will contain: (a) the code used to create the maps, perform the clustering, etc.; (b) instructions on how to download the data used to carry out the redistricting from the US Census Bureau; (c) individual maps per state which show the districts as they are and our solutions; along with (e) links to any other code or materials used.
