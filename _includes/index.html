Welcome

This is the website accompanying the manuscript Gerrymandering and Computational Redistricting by Olivia Guest and Bradley C. Love.

For the materials used, please see the Open Framework Repository (OSF), here: osf.io/8xpq4. This repository will contain: a) the code used to create the maps, perform the clustering, etc.; b) instructions on how to download the data used to carry out the redistricting from the US Census; c) individual maps per state which show the districts as they are and our solutions; d) the corruption scores per state from the Center for Public Integrity; along with e) links to any other code or materials used.

Maps

Below are interactive maps showing the existing districts (left column) and the results from our weighted k-means algorithm (right column). The colors used are to facilitate viewing and are randomly assigned. Moving your cursor over a state highlights its border in black. Clicking on a state zooms both maps in, allowing you to see the state's districts close up for both reality and our solution. To find a specific location click on the magnifying glass, on the top left of both maps, to open up a search box.
